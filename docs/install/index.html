PaperPoint
S3 Reader

中文優先的 M5Paper S3 電子紙閱讀器韌體。

本版功能特色

休眠圖片支援 BMP / JPG / PNG 圖片不必先裁成 540×960；不透明圖片會等比例中央裁切填滿螢幕。
透明 PNG 可疊在閱讀頁上 透明休眠圖可選擇使用目前閱讀頁面或白色背景，非閱讀畫面會自動使用白色背景。
背景快取，關機不等解碼 Paper S3 閒置時會預先建立電子紙灰階快取；若新圖尚未完成，會使用上一張有效快取或內建休眠圖。
中文介面與閱讀字型 預設繁體中文；閱讀內容可使用 SD 卡中的 TTF 字型，系統/UI 字型目前仍不是一般設定項目。
請使用支援 Web Serial 的 Chrome 或 Microsoft Edge 桌面版瀏覽器。 這個安裝頁需要透過 HTTPS 或 localhost 開啟。
  • 用 USB-C 傳輸線接上 M5Stack Paper S3。
  • 長按電源鍵，讓 Paper S3 開機並被電腦偵測。
  • 按下「安裝 / 更新韌體」後，選擇名稱包含 USB JTAG/serial debug unit 的序列埠。
  • 若沒有看到序列埠或連線失敗，請確認使用的是可傳輸資料的 USB-C 線， 重新長按電源鍵開機，或拔插 USB-C 後再試一次。

燒錄完成後的 SD 卡檔案

休眠封面與字型請放在 SD 卡根目錄的指定資料夾。

  • /.sleep/：休眠圖片可放 .bmp、 .jpg、.jpeg、.png。 不必預先裁成 540x960；不透明圖片會中央裁切填滿， 透明 PNG 會完整縮放置中。WEBP、GIF、APNG、CMYK JPEG 與 16-bit PNG 目前不保證支援。
  • /fonts/：閱讀字型可放 .ttf、 .bin、.epdf。其中 .bin/.epdf 檔名需包含尺寸，例如 KingHwaOldSong_17_23x30.bin。
  • 目前設定頁主要調整「閱讀字型」。系統介面/UI 字型還不是一般設定項目， .ttf 也不會拿來當 UI 字型。
