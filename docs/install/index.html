PaperPoint
S3 Reader

中文優先的 M5Paper S3 電子紙閱讀器韌體。

請使用支援 Web Serial 的 Chrome 或 Microsoft Edge 桌面版瀏覽器。 這個安裝頁需要透過 HTTPS 或 localhost 開啟。
  • 用 USB-C 傳輸線接上 M5Stack Paper S3。
  • 長按電源鍵，讓 Paper S3 開機並被電腦偵測。
  • 按下安裝按鈕後，選擇 Paper S3 的序列埠。
  • 若沒有看到序列埠或連線失敗，請確認使用的是可傳輸資料的 USB-C 線， 重新長按電源鍵開機，或拔插 USB-C 後再試一次。

燒錄完成後的 SD 卡檔案

休眠封面與字型請放在 SD 卡根目錄的指定資料夾。

  • /.sleep/：只讀取 .bmp。建議使用 540x960、24-bit 或 32-bit、未壓縮 BMP。不要直接放 JPG、PNG、WEBP；多張 BMP 會隨機顯示。
  • /fonts/：閱讀字型可放 .ttf、 .bin、.epdf。其中 .bin/.epdf 檔名需包含尺寸，例如 KingHwaOldSong_17_23x30.bin。
  • 目前設定頁主要調整「閱讀字型」。系統介面/UI 字型還不是一般設定項目， .ttf 也不會拿來當 UI 字型。
