目前韌體版本：讀取中

中文優先的 M5Stack Paper S3 電子紙閱讀器韌體。

線上燒錄

這個瀏覽器不支援 Web Serial，請改用 Chrome 或 Microsoft Edge。 瀏覽器已拒絕序列埠權限，請確認此頁面是 HTTPS 或 localhost。
  1. 用 USB-C 傳輸線接上 M5Stack Paper S3。
  2. 長按 Paper S3 的電源鍵，讓裝置進入可燒錄狀態。
  3. 按「安裝 / 更新韌體」後選擇序列埠，請選 USB JTAG/serial debug unit 這類名稱的選項。
  4. 如果連線失敗，請拔掉 USB-C、重新長按電源鍵讓裝置進入燒錄狀態，再接回 USB-C 後重新選擇序列埠。

本版重點

  • 1.6.0 加強 Paper S3 直接觸控：首頁、設定分頁、清單與多封面首頁項目都更容易直接點選。
  • 底部導覽改為「返回 / 選擇 / 前頁 / 後頁」，前頁與後頁以整頁清單為單位移動。
  • 新增閱讀狀態列跟隨頁邊距模式，可選擇固定在螢幕底部或跟隨閱讀邊距。
  • 內建雙語 EPUB 使用手冊，瀏覽 /book 時會自動安裝到 SD 卡。
  • 更新內建繁中字型來源與直排標點對齊，閱讀中文字可使用更大的縮放範圍。
  • 精簡內建 ReaderDyslexic、部分 Noto Sans 變體與非英文斷字資料，保留中文閱讀需要的韌體空間。
  • 支援閱讀格式：EPUB / TXT。Wi-Fi 傳輸仍屬未驗證功能。

SD 卡檔案放置

  • /.sleep/：休眠圖可使用 .bmp、.png、.jpg、.jpeg。建議尺寸為 540x960。
  • /fonts/：閱讀字型可使用 .ttf、.bin、.epdf。目前只能調整閱讀字型，不能透過 SD 卡替換系統 UI 字型。
  • 書籍檔可放在 SD 卡任意資料夾，檔案瀏覽器會掃描可閱讀格式。
