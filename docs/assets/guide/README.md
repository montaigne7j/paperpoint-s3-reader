# Guide Screenshots

These images are static reference screenshots for the user guide.  They are intentionally lightweight 540×960 PNGs so the web/manual documentation can show the Paper S3 portrait layout.

For V1.8.0 they remain valid for the main UI structure: home screen, reader menu, reader touch zones, reader value adjustment, and reader settings.  Some footer labels may be shortened by the firmware depending on language, text size, and whether another list page is available.
